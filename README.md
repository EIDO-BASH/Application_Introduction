## Introduction

This project is a personal introduction and CV webpage designed to present my skills, experience, and projects in a clean and modern format.

The page serves as both a digital resume and a portfolio, allowing visitors to learn more about me, explore my work, and easily get in touch.

It is fully responsive and built with a focus on simplicity, accessibility, and good user experience.
